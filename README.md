# Angular E-Commerce Project (work in progess)

This project is currently under development as part of my learning journey in Angular.

## Features
**So far, I have implemented:**
- Product listing from API
- Reusable components (header, product list, etc.)
- Angular services for data handling
- Basic routing between views
- Form handling (e.g. adding products)

**I am currently working on:**
- Seller dashboard for managing products
- CRUD functionality (create, update, delete products)

## Tech Stack
- Angular
- TypeScript
- HTML5
- CSS3
- JSON Server (mock API)

## Purpose

The goal of this project is to gain hands-on experience with Angular and understand how to build structured, scalable frontend applications.

**I applied concepts such as:**

- Components & data binding
- Services & dependency injection
- HTTP requests & API integration
- Form handling

More details and documentation will be added as the project evolves.
